# VzSTORE

Landing page e painel administrativo da loja VzSTORE.

## Estrutura

- `index.html` redireciona para `paginas/index.html`
- `paginas/` contém as páginas públicas e internas do site
- `estilizacoes/` contém o CSS
- `scripts/` contém o JavaScript que controla o login, catálogo e painel da vendedora

## Deploy

Este projeto pode ser publicado no GitHub Pages a partir da branch `main`.
