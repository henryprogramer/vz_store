# VzSTORE - Loja de Roupas Masculina

Site completo de e-commerce para a **LOJA DO VZ**, especializada em moda masculina de qualidade.

## 🎯 Sobre o Projeto

A VzSTORE é uma plataforma online focada em:
- 🔥 **Qualidade**: Peças selecionadas com cuidado
- 🚀 **Entrega**: +7.000 pedidos entregues
- 🇧🇷 **Cobertura**: Enviamos para todo Brasil
- ⏳ **Exclusividade**: Peças limitadas

## 📱 Páginas do Site

- **Início** (`index.html`) - Landing page com destaques da marca
- **Catálogo** (`catalogo.html`) - Vitrine completa de produtos
- **Sobre** (`sobre.html`) - Histórico e missão da loja
- **Contato** (`contato.html`) - Informações de contato e redes sociais
- **Acesso** (`acesso.html`) - Login e cadastro
- **Checkout** (`checkout.html`) - Finalização de compra

## 👥 Áreas de Acesso

### Cliente
- Visualizar catálogo completo
- Montar carrinho de compras
- Acompanhar pedidos
- Histórico de compras

### Vendedora (Admin)
- Painel de operações
- Gestão de produt  os
- Controle de pedidos
- Cadastro de usuários

## 🛠️ Tecnologias

- **HTML5** - Estrutura semântica
- **CSS3** - Estilização responsiva
- **JavaScript Vanilla** - Interatividade sem dependências externas
- **LocalStorage** - Persistência de dados no cliente
